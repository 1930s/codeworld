CodeWorld
 Try Now || Getting Started || Documentation || More Information

CodeWorld helps young programmers like you use mathematics, logic, and computers to express yourself and create your own pictures, animations, and games. You can:

  • Be creative and express yourself

  • Use simple algebra and geometry ideas to create things

  • Build skills to take on more and more impressive projects

  • Build your own technology instead of just using what others give you.

 Getting Started

The quickest way to get started using CodeWorld is to follow the online tutorial.

  1. Follow the link below to use the programming editor.

  2. Press the Help button to get access to the help files.

  3. Click the examples to copy them from the tutorial to your workspace.

  4. Press the Run button to see a program in action.
